## Дневник погоды (Weather Diary)

**Автор:** Никита Онучин

**Описание**

Графическое приложение для ведения личного дневника погоды. Позволяет ежедневно фиксировать погодные условия, добавлять заметки, просматривать историю наблюдений и анализировать изменения. Все данные сохраняются в *JSON*-файл, что обеспечивает их сохранность между запусками программы.

### Возможности
- Ежедневная запись погоды (температура, осадки, облачность).
- Добавление текстовых заметок к каждому дню.
- Просмотр истории наблюдений в виде списка.
- Поиск и фильтрация записей по дате и погодным условиям.
- Сохранение и загрузка данных из *JSON*-файла.
- Интуитивно понятный графический интерфейс.

### Как использовать
1. Убедитесь, что установлен **Python 3.8+**.
2. Скачайте или клонируйте репозиторий.
3. Запустите файл *main.py*.
4. В интерфейсе:
    - Введите дату, выберите погодные параметры и добавьте заметку.
    - Нажмите **«Сохранить запись»** для добавления новой записи в дневник.
    - Используйте фильтры по дате и погоде для поиска нужных наблюдений.
    - Просматривайте историю в специальном окне.
    - Сохраняйте историю или очищайте её при необходимости.

### Примеры использования (тест-кейсы)

| **Действие** | **Ожидаемый результат** |
| :--- | :--- |
| Ввести данные за сегодня и нажать «Сохранить запись» | Новая запись появляется в истории |
| Выбрать фильтр по дате и нажать «Поиск» | Отображаются записи только за выбранный день |
| Просмотреть историю | В окне истории отображается список всех сохранённых записей |
| Закрыть и открыть приложение | История загружается из *weather_data.json* |
