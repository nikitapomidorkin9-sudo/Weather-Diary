# Weather Diary (Прогноз погоды)

## Автор
Никита Онучин

## Описание
Графическое приложение для учета личных расходов. Позволяет добавить транзакции,фильтровать их по категории и периоду,автомотически подсчитать
промежуток времени.Данные сохраняются в JSON-файл,что гарантирует сохранность информации между запусками.
##  Как использовать
1. Убедитесь, что установлен **Python 3.8+**.
2. Скачайте или клонируйте репозиторий.
3. Запустите в приложении программу ExpenseTracker
4. В интерфейсе:
   - Введите сумму, выберите категорию, укажите дату в формате `ГГГГ-ММ-ДД`.
   - Нажмите **"Добавить расход"**.
   - Для анализа используйте фильтр и кнопку **"Применить фильтр"**. Сумма пересчитается автоматически.
 
##  Примеры использования (Тест-кейсы)
| Действие | Ожидаемый результат |
|----------|---------------------|
| Ввести Температура `-` | Ошибка: "Температура должна быть положительным числом или отрицательным." |
| Ввести дату `20/05/2024` | Ошибка: "Неверный формат даты. Используйте ГГГГ-ММ-ДД." |
| Добавить: `20.05.2024`, `20`, `солнечно` | Запись появляется в таблице, сохраняется в `expenses.json` |
| Фильтр: Категория=`описание`, Период=`2024-05-01` > `2024-05-31` | В таблице  погода в мае|
| Закрыть и открыть приложение | Все записи загружаются из JSON автоматически |
